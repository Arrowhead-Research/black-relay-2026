# Black Relay 2026 Landing Page Prototype

Static prototype based on `PRD.md`.

## View locally

Open `index.html` directly in a browser, or serve the directory:

```bash
python3 -m http.server 8080
```

Then visit <http://localhost:8080>.

## Configure CTA links

Edit `CTA_LINKS` in `site.js` when the official application, advisor, or contact URLs are available.

## Files

- `index.html` — semantic single-page landing page
- `styles.css` — responsive dark technical visual design
- `site.js` — CTA constants and mobile navigation
- `PRD.md` — source requirements
